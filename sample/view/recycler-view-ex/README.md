# Summary
RecyclerView timer example

## Preview

2개의 RecyclerView를 활용하고 Coroutines을 활용하여 2개의 목록을 관리

- A 리스트에는 10개의 아이템을 노출
- B 리스트에는 A 목록에서 넘어온 데이터를 기반으로 노출하며 5초 후 A 리스트로 넘어가는 처리

coroutines을 활용하여 각각의 아이템을 처리하는 샘플
